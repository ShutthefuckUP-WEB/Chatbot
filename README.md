# Basic Chatbot

A simple rule-based chatbot built using Python. The chatbot responds to predefined user inputs such as greetings and basic questions.

## Features
- Responds to basic greetings
- Uses predefined replies
- Runs in the Python console
- Continues chatting until the user types "bye"
- Beginner-friendly and easy to understand

## Requirements
- Python 3

## How to Run

1. Download or clone this repository.
2. Open a terminal in the project folder.
3. Run the following command:

```bash
python chatbot.py
```

## Example

```
Welcome to the Basic Chatbot!

You: hello
Bot: Hi!

You: how are you
Bot: I'm fine, thanks! How about you?

You: bye
Bot: Goodbye! Have a nice day!
```
